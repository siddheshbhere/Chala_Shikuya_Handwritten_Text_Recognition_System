Instructions for How to Use the website

  1. The first time a user visits the site, he or she must create an account by clicking on the register/login option at the top right corner. The user must log in after registering.
  2. User will be redirected to next page where he or she will be able to select the language.
  3. After choosing a language, the user can either practise on the canvas or watch video tutorials.
  4. Once the preceding step is completed, the user will have access to the alphabets and numbers sections.
